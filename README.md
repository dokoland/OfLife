# OfLife

A personal playground for experimenting with Conway's Game of Life.

This project started as a way to explore different implementation strategies, performance optimizations and data structures while keeping the code clean and easy to understand.

Rather than being a complete Game of Life application, **OfLife** is a playground for trying out ideas.

## Goals

- Explore different implementations of Conway's Game of Life
- Experiment with performance optimizations
- Compare sparse and dense world representations
- Keep the code simple and maintainable
- Have fun experimenting

## Why?

Game of Life is a fascinating example of how simple rules can create surprisingly complex behaviour.

This project was created to experiment with algorithms and software design rather than to build yet another Game of Life implementation.

## Build

Open the solution in Visual Studio and run the project.
